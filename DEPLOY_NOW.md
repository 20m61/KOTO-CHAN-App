## 📋 **手動デプロイ手順**

### 1. **Vercelアクセス**
```
https://vercel.com/new/git
```

### 2. **リポジトリ選択**
- **GitHub**: `20m61/KOTO-CHAN-App`
- **Import**: クリック

### 3. **プロジェクト設定**
```

Framework: Next.js (自動検出)
Root Directory: ./
```

### 4. **環境変数設定**
